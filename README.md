# 📖 Overview

VectraOS is a premium AI-powered enterprise automation platform designed to simplify business workflows through intelligent process automation, real-time insights, and scalable user experiences.

Built as a modern SaaS landing page, the project demonstrates production-ready frontend engineering with responsive layouts, semantic HTML, SEO optimization, reusable components, and high-performance rendering.

The application showcases enterprise-grade UI/UX principles while emphasizing accessibility, maintainability, and clean component architecture.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/suraj-5021/vectraos-ai-automation-platform.git
```

Navigate to the project

```bash
cd vectraos-ai-automation-platform
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Create production build

```bash
npm run build
```
